# Oxygen and sulfur define a shared local geometry in multivariable periodic table space

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19634706.svg)](https://doi.org/10.5281/zenodo.19634706)

This repository contains the notebook, figures, dataset, and PDF associated with the preprint:

"Oxygen and sulfur define a shared local geometry in multivariable periodic table space"

The analysis uses a multivariable representation of the 118-element periodic table based on:

- electronegativity
- electron affinity
- first ionization energy
- oxygen-affinity proxy
- redox-flexibility proxy
- group
- period

Principal component analysis (PCA) was used to construct shared 2D and 3D spaces, and all elements were ranked according to Euclidean distance from oxygen.

Main finding:

Sulfur reproduced the oxygen-centered local neighborhood under the tested representation and generated the most compact local geometry.

## Repository structure
